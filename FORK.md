# Fork Attribution

This repository, `rivenai/riven-auth-source`, is a Riven Holdings fork of
[Logto](https://github.com/logto-io/logto) at release **v1.39.0**.

Logto is an open-source identity infrastructure project released under the
[Mozilla Public License 2.0 (MPL 2.0)](LICENSE). The upstream source is available
at <https://github.com/logto-io/logto>.

This fork (`Riven Auth`) applies Riven-specific branding, packaging, and
operational patches so that Logto can be shipped as the Riven identity platform
while remaining compliant with the upstream license. All upstream license and
attribution notices are preserved.
